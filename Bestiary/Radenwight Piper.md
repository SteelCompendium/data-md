|                                        |                                |
| :------------------------------------- | :----------------------------- |
| Keywords: Attack, Magic, Melee, Ranged | Type: Action                   |

|                         |                        |
| :---------------------- | :--------------------- |
| Keywords: Melee, Weapon | Type: Triggered Action |

Trigger: An ally deals damage to the target.

Effect: The piper makes a [[Free Strike|free strike]] against the target.

**Musical Suggestion**
